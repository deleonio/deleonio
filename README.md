> I build **accessible, sovereign UI** for the public sector — and the tooling that keeps big open-source repos maintainable.

---

## 🧭 What I'm about

- ♿ **Accessibility-first** — BITV 2.0 / WCAG as a default, not an afterthought
- 🇪🇺 **Digital sovereignty** — European-native, EUPL-licensed open source
- 🧩 **Web Components over lock-in** — framework-agnostic by design (built on Stencil)
- 🔐 **Secure supply chains** — SBOM, SLSA, DevSecOps as standard practice
- 🤖 **Automating the boring parts** — agentic routines for issue triage, PRs & releases

---

## 🚀 What I'm building

| Project | What it is |
|---|---|
| **[KoliBri — @public-ui](https://github.com/public-ui/kolibri)** | The accessible HTML standard. A BITV-2.0 / WCAG-compliant, framework-agnostic Web Component library — the human-facing UX layer of the Deutschland-Stack. `EUPL-1.2` |
| **[leanup — @leanupjs](https://github.com/leanupjs/leanup)** | Framework-independent CLI & toolkit to develop, test and build SPAs / PWAs. `TypeScript` |
| **[StackAtlas](https://stackatlas.dev)** | A reference atlas for the modern tech stack — explore, compare and roadmap technologies and architecture choices. |
| **[DiPA](https://github.com/DiPA-Projekt)** | Public-sector digital records project — accessible, sovereign, open by default. |
